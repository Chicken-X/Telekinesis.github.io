Q3Telekinesis

Quake 3 Arena mod

Telekinesis mod for Quake 3 Arena that adds powerful mind-based abilities to the game.

Manipulate objects, throw enemies, and master the force in intense arena combat. This mod introduces a completely new gameplay mechanic with telekinetic powers that can be used in both single-player and multiplayer modes.

Q3Telekinesis Installation

  • Extract the downloaded .pk3 file to your Quake 3/baseq3/ folder
  • Launch Quake 3 Arena
  • Go to Mods menu and select "Q3Telekinesis"
  • Enjoy telekinetic combat!
